# 山大王脚手架


## Getting Started

安装：

```bash
npm install -g @sdw-cli/core
# OR
yarn global add @sdw-cli/core
```

创建项目

```bash
sdw-cli init 
```

发布项目

```bash
sdw-cli publish
```

## More

DEBUG 模式：

```bash
sdw-cli --debug
```

指定本地包：

```bash
sdw-cli init --packagePath /Users/sam/Desktop/sdw-cli/packages/init/
```
